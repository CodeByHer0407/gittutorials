## 📌 About This Repo

This repo is my structured documentation of backend engineering concepts — written as I learn them on the job and in my own time.

Each topic folder contains:
- **Concept notes** — what it is, why it matters
- **Code examples** — minimal, working, well-commented
- **Real-world context** — how it's used in production

The goal is a reference I can use to revise quickly and demonstrate depth to anyone reviewing my work.
